              Description

              I'm the maintainer of qdox for Fedora Linux and we are building everything from source. As a result I need to do svn export of every tag and create custom tarballs. This has a number on inconviniences like nobody is able to verify whether these are the exact sources used for the official qdox release and etc.
              Please tell whether this is something you would like to provide as an additional download option?
              If yes but you don't want to spend time on it I'll try to propose a patch for the maven build to produce them.
